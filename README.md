# Deep Learning & Artificial Neural Networks (ANN)

## Description
Built a Deep Learning model utilizing Artificial Neural Networks to simulate human-like data processing. This project involved designing network architecture, managing layers, and optimizing training processes to handle complex non-linear data patterns.

## Tools used:
- **Language:** Python
- **Libraries:** TensorFlow/Keras, NumPy, Pandas.

## Key Features:
- Designing Multi-layer Neural Networks.
- Activation Functions & Optimization.
- Handling complex non-linear datasets.
